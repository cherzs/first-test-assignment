# Frontend Component - Test Assignment

Modal dialog component with page selection interface, built using HTML, CSS, and JavaScript.

## Design Source

[Figma Design Link](https://www.figma.com/design/dwBFtlKY933OJWWSrGPs5q/Frontend?node-id=0-1)

## Getting Started

1. Clone repository:
```bash
git clone https://github.com/cherzs/first-test-assignment.git
cd first-test-assignment
```

2. Open `index.html` in browser, or run local server:
```bash
python3 -m http.server 8000
```

3. Access `http://localhost:8000`

## Features

- Modal dialog with page selection
- Interactive checkboxes with state management
- "Done" button with toast notification
- Responsive design

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
